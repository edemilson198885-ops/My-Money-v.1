# My Money - pacote pronto para postar

## Esta versão já vem com cache busting
Versão atual: `1.0.5`

## Como publicar
1. Suba tudo mantendo a estrutura:
   - `index.html`
   - `manifest.json`
   - `assets/`
2. Faça commit.
3. GitHub Pages: branch `main`, pasta `/(root)`.

## Como atualizar no futuro
Mude a versão em:
- `index.html` (`window.APP_VERSION`)
- `assets/js/config.js` (`APP_VERSION`)

Exemplo: `1.0.5` -> `1.0.6`
